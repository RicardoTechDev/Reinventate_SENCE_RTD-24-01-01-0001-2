Dos opciones 
- Crear repo en github y hacer git clone
	crear repo remoto en github.com
	abrir consola gitbash en local donde voy a clonar repo
	git clone https://github.com/........nombre usuario/repo.git
	en la consola cd nombre repo clonado, al ingresar se activa git
	git pull para traer los cambios en remoto
	
- crear repo local, luego crear repo en github
	AL INICIAR UN PROYECTO
    	git init
    	git add .
    	git commit -m "Mensaje"
    	git remote add origin url-de-github
    	git push -u origin branch

REALICEMOS CAMBIOS
    git status (para verificar si hya cambios  pendientes)
    git add .
    git commit -m "Mensaje"
    git push -u origin branch

BRANCHES
    git checkout -b nombre-de-tu-rama

MERGE
    git checkout master
    git pull origin master
    git merge nombre-de-tu-rama

ELIMINAR RAMAS
    git branch -d nombre-de-tu-rama       # Elimina localmente
    git push origin --delete nombre-de-tu-rama  # Elimina remotamente

    	git remote add origin url-de-github

REALICEMOS CAMBIOS
    git status (para verificar si hya cambios  pendientes)
    git add .

BRANCHES (Ramas)
    git branch   (para ver en que rama estoy parado) 
    git branch nombre-nueva-rama (crea una nueva rama)
    git branch -m nombre-actual-rama nuevo-nombre-rama  (nos permite cambiar el nombre de una rama)
    git checkout nombre-rama (nos permite cambiar de la rama actual a otra rama)
    git checkout -b nombre-nueva-rama (Crea una nueva y se mueve a ella)
    git fetch (nos permite traer todos los cambios de mi proyecto remoto inlcuidas las nuevas ramas)



MERGE (Unificar ramas en una s√≥la)
    git checkout master
    git pull origin master
    git merge nombre-de-tu-rama  (Debo estar parado en la rama que va a recibir los cambios)

ELIMINAR RAMAS
    git branch -d nombre-de-tu-rama       # Elimina localmente
    git push origin --delete nombre-de-tu-rama  # Elimina remotamente
